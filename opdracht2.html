CSS / HTML The Position Planet Challenge

leerdoel:

Je leert in deze oefening hoe je via CSS met de 'position' eigenschappen omgaat. Je leert elementen te plaatsen met zowel 'absolute' als 'relative' eigenschappen.

Opdracht:

Plaats de verschillende elementen zoals in de volgende afbeelding. Doe dit zo nauwkeurig mogelijk!

  • Let goed op de borders van de elementen. In het voorbeeld overlappen deze elkaar niet, zorg daar ook voor in jouw oplossing.
  • Tip: bepaal aan de hand van de width en de height van de diverse elementen hun positie
